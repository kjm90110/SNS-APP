프론트엔드 + Node.js API 연동 과제 안내 (JWT 인증 필수)
여러분이 만든 Node.js 백엔드 API를 기반으로, HTML + CSS + JavaScript만 사용하여 브라우저용 프론트엔드 화면을 직접 구현하는 과제입니다.
백엔드 API를 실제로 호출하며 작동하는 “완전한 웹 서비스 흐름”을 구축합니다.
모든 게시글 관련 기능(조회 / 작성 / 수정 / 삭제)는 반드시 로그인하여 발급받은 JWT를 Authorization 헤더에 담아 요청해야만 동작합니다.
